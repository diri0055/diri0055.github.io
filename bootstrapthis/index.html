About Bootstrap

Bootstrap is an open source toolkit for developing with HTML, CSS and JS. You can build your entire website with Bootstrap. It has responsive grid system and extensive prebuilt powerful components.

Flex-box grid layout

Bootstrap has powerful responsive flex-box grid layout for all devices. You can build your entire website how ever you want thanks to the 12 column system.
